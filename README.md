# Leather Jacket Care Tips

To keep your sheepskin bomber jacket looking new:

- Store it in a cool, dry place.
- Avoid prolonged exposure to direct sunlight.
- Clean using leather-specific products.
- Never machine wash genuine leather.
- Hang on a wide wooden hanger to maintain shape.
